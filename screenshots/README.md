# Screenshots

This folder contains screenshots of the Alarm Master app.

## How to Take Screenshots

### On iOS Simulator:
1. Open the app in Xcode Simulator
2. Navigate to the screen you want to capture
3. Press `Cmd + S` to save a screenshot, or
4. Use `Device > Screenshot` from the menu

### On Physical Device:
1. Open the app on your iPhone/iPad
2. Navigate to the screen you want to capture
3. Press `Volume Up + Power Button` (or `Side Button + Volume Up` on newer iPhones) simultaneously
4. Screenshots are saved to your Photos app
5. Transfer them to your Mac and place them in this folder

## Recommended Screenshots

Consider capturing:
- Main screen with weather bar and alarm list
- Empty state (no alarms)
- Add alarm screen
- Active alarm screen (when alarm is ringing)
- Master toggle in both ON and OFF states
- Weather bar with different weather conditions

## File Naming

Suggested naming convention:
- `main-screen.png` - Main screen with alarms
- `empty-state.png` - Empty state view
- `add-alarm.png` - Add alarm screen
- `active-alarm.png` - Alarm ringing screen
- `weather-bar.png` - Weather bar detail
